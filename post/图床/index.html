免费图床

s：推荐顺序由高到低

GitHub 图床的正确用法，通过 jsDelivr CDN 全球加速

1:公益图床

长期保存需要注册使用，这个图床服务器在国内应该还用了cdn，总之速度非常快

2. 七牛云

简介：注册认证以后会有10g免费空间，国内外流量个10g，插件支持的多，有免费ssl证书，https流量收费。无图片上传限制。

注意：其三十天以后会收回测试域名，需要自己绑定已备案的域名。

3.又拍云

简介：注册认证后有10G永久免费空间，每月15G的HTTP和HTTPS流量，提供两款可以免费续期的SSL证书，不过用户需要加入又拍云联盟（即在网站底部添加又拍云logo及官网链接）

注意：需要绑定自己的已备案域名，又拍云认证比较麻烦，建议使用七牛云

图片上传限制：无
